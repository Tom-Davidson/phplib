phplib
======

Library of useful (mostly) stand-alone php components.

To Use:
======
"repositories": [
...
    {
        "type": "git",
        "url":  "git@github.com:me/my-repo.git"
    }
...
],
"require": {
...
    "me/my-repo": "*"
...
}

You might also need to add:
	"minimum-stability": "dev"
